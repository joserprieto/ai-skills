#### .prettierignore — Files Excluded from Formatting

```text
CHANGELOG.md
AGENTS.md
CLAUDE.md
CLAUDE.local.md
*.local.*
node_modules/
.venv/
dist/
build/
*.hbs
```

**Why these exclusions:**

- **`CHANGELOG.md`** — Auto-generated by `commit-and-tag-version`. Formatting it would create
  conflicts on every release because prettier would rewrite the generated output differently than
  the templates produce.
- **`*.hbs`** — Handlebars templates use whitespace-sensitive `{{~` trim markers. Prettier would
  add/remove whitespace that changes the rendered changelog output.
- **`AGENTS.md`, `CLAUDE.md`, `CLAUDE.local.md`** — Local agent instruction files that are not
  versioned. Excluding them avoids noise from files that don't go through CI.
