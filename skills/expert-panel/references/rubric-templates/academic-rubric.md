# Academic Evaluation Rubric

Use when evaluating research papers, technical whitepapers, academic proposals, or scientific
reports.

## Criteria

### 1. Scientific rigour (weight: 35%)

| Criterion                   | Description                                                  |
| --------------------------- | ------------------------------------------------------------ |
| Methodology soundness       | Is the research methodology appropriate and well-designed?   |
| Statistical validity        | Are statistical methods correctly applied and interpreted?   |
| Reproducibility             | Could another researcher reproduce the study from the paper? |
| Limitations acknowledgement | Are limitations honestly discussed?                          |

### 2. Contribution (weight: 30%)

| Criterion               | Description                                     |
| ----------------------- | ----------------------------------------------- |
| Novelty                 | Does it advance the state of the art?           |
| Significance            | How important is the contribution to the field? |
| Practical applicability | Can the findings be applied in practice?        |
| Scope appropriateness   | Is the scope well-defined and achievable?       |

### 3. Presentation (weight: 20%)

| Criterion                | Description                                           |
| ------------------------ | ----------------------------------------------------- |
| Writing quality          | Is the paper well-written and clearly structured?     |
| Figure and table quality | Are visualisations informative and properly labelled? |
| Related work coverage    | Is the literature review comprehensive and fair?      |
| Abstract accuracy        | Does the abstract accurately represent the content?   |

### 4. Ethics and integrity (weight: 15%)

| Criterion              | Description                                        |
| ---------------------- | -------------------------------------------------- |
| Citation integrity     | Are sources properly cited and attributed?         |
| Data transparency      | Is data available or its unavailability justified? |
| Conflict of interest   | Are potential conflicts disclosed?                 |
| Ethical considerations | Are ethical implications discussed where relevant? |

## Scoring scale

| Score | Label               | Description                                                     |
| ----- | ------------------- | --------------------------------------------------------------- |
| 1-2   | Reject              | Fundamental flaws. Not suitable for publication.                |
| 3-4   | Major revision      | Significant issues requiring substantial rework.                |
| 5-6   | Minor revision      | Sound work with issues that can be addressed.                   |
| 7-8   | Accept with changes | Good work. Minor changes recommended.                           |
| 9-10  | Strong accept       | Excellent contribution. Publish as-is or with cosmetic changes. |
