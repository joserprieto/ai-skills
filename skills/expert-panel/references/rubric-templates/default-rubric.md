# Default Evaluation Rubric

General-purpose rubric suitable for most artifact evaluations. Modify criteria as needed for the
specific evaluation context.

## Criteria

### 1. Quality (weight: 40%)

| Criterion     | Description                                                                       |
| ------------- | --------------------------------------------------------------------------------- |
| Accuracy      | Are facts, data, claims, and technical details correct?                           |
| Completeness  | Does the artifact cover everything within its stated scope?                       |
| Clarity       | Is the artifact clear, well-organised, and understandable by its target audience? |
| Actionability | Are conclusions, recommendations, or outputs concrete and usable?                 |

### 2. Methodology (weight: 30%)

| Criterion         | Description                                                           |
| ----------------- | --------------------------------------------------------------------- |
| Process adherence | Were appropriate standards, procedures, or best practices followed?   |
| Documentation     | Are methods, tools, and decisions documented and transparent?         |
| Reproducibility   | Could another professional reproduce the work from the documentation? |
| Evidence quality  | Are claims supported by verifiable evidence?                          |

### 3. Value (weight: 30%)

| Criterion            | Description                                                              |
| -------------------- | ------------------------------------------------------------------------ |
| Fitness for purpose  | Does the artifact serve its intended purpose effectively?                |
| Cost proportionality | Was the effort/cost proportionate to the value delivered?                |
| Timeliness           | Was the artifact delivered within a reasonable timeframe?                |
| Novelty              | Does the artifact contribute new insights beyond what was already known? |

## Scoring scale

| Score | Label              | Description                                                  |
| ----- | ------------------ | ------------------------------------------------------------ |
| 1-2   | Deficient          | Fundamental gaps or errors. Not usable without major rework. |
| 3-4   | Below expectations | Significant issues. Requires substantial revision.           |
| 5-6   | Adequate           | Meets minimum requirements but has notable gaps.             |
| 7-8   | Good               | Solid work with minor issues. Fit for purpose.               |
| 9-10  | Excellent          | Exceptional quality. Sets a high standard.                   |

## Usage

Each evaluator scores every criterion on the 1-10 scale and provides written justification. The
consolidation agent aggregates scores by criterion, area, and overall panel average.

Weights are used during consolidation to compute weighted averages. Individual evaluators score
criteria without applying weights — they just score what they see.
