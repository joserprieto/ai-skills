# Panel Template: Security Assessment

Use this template when evaluating security reports, incident response reports, penetration test
results, vulnerability assessments, or security audit findings.

## Suggested areas and evaluators

### Area 1: Digital Forensics and Incident Response (3-5 evaluators)

| ID     | Specialisation                             | Focus                                             |
| ------ | ------------------------------------------ | ------------------------------------------------- |
| DFIR-1 | Chain of custody and evidence preservation | Were forensic procedures followed?                |
| DFIR-2 | Container/cloud forensics                  | Container, VM, cloud-specific forensic techniques |
| DFIR-3 | Log analysis and timeline reconstruction   | Is the timeline accurate and complete?            |
| DFIR-4 | Malware analysis and IOC identification    | Were IOCs properly identified and documented?     |
| DFIR-5 | DFIR methodology compliance                | NIST SP 800-86, ISO 27037 adherence               |

### Area 2: Cybersecurity and vulnerability assessment (3-5 evaluators)

| ID    | Specialisation                            | Focus                                     |
| ----- | ----------------------------------------- | ----------------------------------------- |
| SEC-1 | CVE analysis and exploit chain validation | Is the vulnerability analysis correct?    |
| SEC-2 | Network security and lateral movement     | Were network implications fully assessed? |
| SEC-3 | Credential exposure and identity security | Were credential risks properly evaluated? |
| SEC-4 | Container/cloud security and hardening    | Were platform-specific risks addressed?   |
| SEC-5 | OSINT and threat intelligence             | Were threat actors properly attributed?   |

### Area 3: IR engagement quality and professionalism (3-5 evaluators)

| ID   | Specialisation                                  | Focus                                         |
| ---- | ----------------------------------------------- | --------------------------------------------- |
| QA-1 | Report writing standards and clarity            | Is the report well-written and clear?         |
| QA-2 | Client communication and stakeholder management | Was the engagement well-managed?              |
| QA-3 | Time-to-value and efficiency                    | Was the work proportionate to the cost?       |
| QA-4 | Methodology transparency and reproducibility    | Could another analyst reproduce the findings? |
| QA-5 | Professional ethics and source attribution      | Were findings properly attributed?            |

## Suggested minimum panel

For a quick review: DFIR-1, SEC-1, SEC-2, QA-1, QA-3 (5 evaluators, 1 batch)

For a thorough review: All evaluators above (15 evaluators, 3 batches)
